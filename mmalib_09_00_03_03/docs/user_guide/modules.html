 ▼MMALIB_CNN_convolve_col_smallNo_legacy_ixX_ixX_oxX[DEPRECATED] Kernel for computing CNN-style 2D convolution using column major data ordering on the input and output feature maps. This approach computes more quickly if filter grouping is chosen such that Ni=No=1, or if filter grouping is chosen such that NiFrFc < MMA_SIZE, otherwise use regular convolution method MMALIB_CNN_convolve_row_ixX_ixX_oxX. This kernel is also referred to as depth-wise convolution.
THIS KERNEL EXISTS TO EASE TRANSITION OF CODE FROM C7100 TO C7120 AND IMPLEMENTS THE SAME ALGORITHM AS WAS USED IN C7100 ON C7120. HOWEVER, THIS IS NOT THE MOST EFFICIENT USE OF C7120 AND IT IS RECOMMENDED TO USE MMALIB_CNN_convolve_col_smallNo_ixX_ixX_oxX OR MMALIB_CNN_convolve_col_smallNo_highPrecision AS APPROPRIATE. This kernel will be removed in future releases
