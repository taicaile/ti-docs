Detailed Description

PMIC Miscellaneous control param Configuration Note: validParams is input param for all Set and Get APIs. other params except validParams is input param for Set APIs and output param for Get APIs.

Parameters
validParamsSelection of structure parameters to be set, from the combination of Pmic_MiscCtrlValidParamCfg and the corresponding member value must be updated. Valid values Pmic_MiscCtrlValidParamCfg
amuxOutRefOutEnEnable/Disable Band gap Voltage to AMUX OUT Pin (for TPS6594x Leo Device) or RFF OUT Pin (for LP8764x Leo Device) Valid only when PMIC_CFG_AMUX_OUT_REF_OUT_EN_VALID bit is set Valid values Pmic_Tps6594xLeo_AMuxOutPinCtrl_Cfg (for TPS6594x Leo Device) Pmic_Lp8764xHera_RefOutPinCtrl_Cfg (for LP8764x Hera Device)
clkMonEnEnable or Disable internal Clock Monitoring Valid only when PMIC_CFG_CLK_MON_EN_VALID bit is set. Valid values Pmic_InternalClkMonitor_Cfg
syncClkOutFreqSelSelects SYNCCLKOUT Frequency Valid values Pmic_SyncClkOut_Freq_Sel Valid only when PMIC_CFG_SYNC_CLK_OUT_FREQ_SEL_VALID bit is set
extClkSelExternal clock Selection Valid values Pmic_ExtClk_Sel Valid only when PMIC_CFG_EXT_CLK_SEL_VALID bit is set.
syncClkInFreqSelects External clock Frequency Valid only when PMIC_CFG_SYNC_CLK_IN_FREQ_VALID bit is set. Valid values Pmic_Tps6594xLeo_ExtClk_Freq_Sel (for TPS6594x Leo Device) Pmic_Lp8764xHera_ExtClk_Freq_Sel (for LP8764x Hera Device)
nRstOutSocSignalConfigure NRSTOUT_SOC Signal Note: When Application configures nRstOutSocSignal as PMIC_PIN_SIGNAL_LEVEL_LOW then SOC will be in reset Valid only when PMIC_CFG_NRSTOUT_SOC_VALID bit is set. Valid values Pmic_SignalLvl
nRstOutSignalConfigure NRSTOUT Signal Note: When Application configures nRstOutSignal as PMIC_PIN_SIGNAL_LEVEL_LOW then MCU will be in reset Valid only when PMIC_CFG_NRSTOUT_VALID bit is set. Valid values Pmic_SignalLvl
