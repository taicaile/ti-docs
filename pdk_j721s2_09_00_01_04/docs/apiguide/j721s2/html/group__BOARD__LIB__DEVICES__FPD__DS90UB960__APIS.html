Get ub960 i2c address.

This function is used to get the I2C address and channel details of UB960 module.

Parameters
chNum[IN] buffer to hold i2c channel number
i2cAddr[IN] buffer to hold i2c address
csiInst[IN] CSIRX Instance See BOARD_csiInst for more details.
Returns
In case of wrong parameters, chNum and i2cAddr are sent back as '0'.
