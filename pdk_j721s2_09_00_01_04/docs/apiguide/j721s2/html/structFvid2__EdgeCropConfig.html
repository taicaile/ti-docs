Fvid2_EdgeCropConfig Struct Reference

Detailed Description

Structure containing edge crop configuration - used in DSS.

struct Fvid2_EdgeCropConfig A frame has four edges: top, bottom, left and right. User can decide to crop in any direction. cropTop will be used to remove the particular number of lines from the top of the frame. Similarly other fields will be used.
