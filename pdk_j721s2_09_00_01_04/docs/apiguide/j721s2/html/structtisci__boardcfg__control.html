tisci_boardcfg_control Struct Reference
SCIClient Driver » Texas Instruments System Controller Interface » tisci_boardcfg

Detailed Description

Used to enable/disable features in DMSC based on usecase.

Parameters
subhdrMagic and size for integrity check.
main_isolation_enableEnable/disable support for DMSC main isolation. If disabled, main isolation SCI message will be rejected with NAK.
main_isolation_hostidHost-ID allowed to send SCI-message for main isolation. If mismatch, SCI message will be rejected with NAK.
