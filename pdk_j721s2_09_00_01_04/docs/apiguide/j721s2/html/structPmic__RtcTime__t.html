Pmic_RtcTime_t Struct Reference
PMIC Driver » PMIC RTC Driver API

Detailed Description

RTC time configuration. The Pmic_RtcTime_s structure contains set of time parameters to set/get the RTC time.

Parameters
validParamsValidate params Bits. Depending on the parameters want to get/set, corresponding bits should be set in validParam. For valid values: Pmic_RtcTimeValidParamBits
secondsValue to represent Seconds. Valid only when PMIC_RTC_TIME_CFG_SEC_VALID bit of validParams is set.
minutesValue to represent Minutes. Valid only when PMIC_RTC_TIME_CFG_MIN_VALID bit of validParams is set.
hourValue to represent Hours. Valid only when PMIC_RTC_TIME_CFG_HRS_VALID bit of validParams is set.
timeModeValue to represent Time Mode. For valid values: Pmic_RtcTimeMode. Valid only when PMIC_RTC_TIME_CFG_TIMEMODE_VALID bit of validParams is set.
meridianModeValue to represent Maridian Mode. For valid values: Pmic_RtcMeridienMode. Valid only when PMIC_RTC_TIME_CFG_MERIDIAN_VALID bit of validParams is set.
