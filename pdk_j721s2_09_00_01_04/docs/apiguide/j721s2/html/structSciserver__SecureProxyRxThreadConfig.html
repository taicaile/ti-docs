Sciserver_SecureProxyRxThreadConfig Struct Reference
SCIClient Driver » Sciserver SecProxy Description

Detailed Description

Describes the Secure proxy Reader (Rx) thread configuration.

Parameters
host_id- Host ID that will use this thread
thread_id- Secure proxy thread ID
evt_threshold- Event ID to generate for threshold event (optional)
evt_error- Event ID to generate in case of thread error (optional)
ring_config_instance_id- Which ring does this thread use? NOTE: There should be only a single Rx thread per ring, though multiple Tx threads may be associated with the same Rx thread's ring. This is an index into soc_ring_configurations
threshold- Threshold for events (optional)
secure_proxy_instance_id- Which secure proxy instance does this thread belong to? This is an index into gSciserverSecureProxyInstances
