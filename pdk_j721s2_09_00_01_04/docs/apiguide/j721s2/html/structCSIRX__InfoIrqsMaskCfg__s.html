Detailed Description

Information interrupt mask. Bit addressable mask register in order to independently enable each event to trigger the InfoIrq line. Only events whose corresponding bit is set to 1 can trigger the interruption line. Hard reset is 0x00000000, i.e. interrupt line disabled
