Pmic_CommonCtrlStat_t Struct Reference
PMIC Driver » PMIC Common Driver API

Detailed Description

PMIC common control param status Note: validParams is input param for all Get APIs. other params except validParams is output param for Get APIs.

Parameters
validParamsSelection of structure parameters to be set, from the combination of Pmic_CommonCtrlStatValidParamCfg and the corresponding member value must be updated Valid values Pmic_CommonCtrlStatValidParamCfg
spmiLpmStatSPMI Low Power Mode Control Status. Valid only when PMIC_CFG_SPMI_LPM_STAT_VALID bit is set. Valid values Pmic_SpmiLpmModeCtrl_Stat
forceEnDrvLowStatStatus of ENABLE_DRV Configuration by I2C/SPI Valid only when PMIC_CFG_FORCE_ENABLE_DRV_LOW_STAT_VALID bit is set. Valid values Pmic_EnableDrvI2CSPICfg_Stat
bbEocIndicationBackup Battery End of charge Indication Status Valid only when PMIC_CFG_BB_EOC_INDICATION_STAT_VALID bit is set Valid only for TPS6594x Leo Device Valid values Pmic_Tps6594xLeo_BBEoCIndicationStat
regLockStatRegister lock status Valid only when PMIC_CFG_REGISTER_LOCK_STAT_VALID bit is set Valid values Pmic_RegisterLock_Stat
extClkValidityExternal clock validity status. The status value is valid only when External clock is connected Ignore the status value when External clock is not connected Valid only when PMIC_CFG_EXT_CLK_VALIDITY_STAT_VALID bit is set Valid values Pmic_ExtClkValidStat
startupPinStartup(nPWRON/Enable) pin status Valid only when PMIC_CFG_STARTUP_PIN_STAT_VALID bit is set Valid values Pmic_SignalLvl
enDrvPinEN_DRV Pin status Valid only when PMIC_CFG_EN_DRV_PIN_STAT_VALID bit is set. Valid values Pmic_SignalLvl
nRstOutSocPinnRSTOUT_SOC Pin status Valid only when PMIC_CFG_NRSTOUTSOC_PIN_STAT_VALID bit is set. Valid values Pmic_SignalLvl
nRstOutPinnRSTOUT Pin status Valid only when PMIC_CFG_NRSTOUT_PIN_STAT_VALID bit is set. Valid values Pmic_SignalLvl
nIntPinnINT Pin status Valid only when PMIC_CFG_NINT_PIN_STAT_VALID bit is set. Valid values Pmic_SignalLvl
