◆ MailboxConfigIdleMode()

void MailboxConfigIdleMode ( uintptr_t  baseAddr,
uint32_t  idleMode 
)

This function configures the idle mode of the mailbox.

Parameters
baseAddrIt is the Memory address of the Mailbox instance.
idleModeIdle mode to be configured. Possible values are 0x0: Force-idle. An idle request is acknowledged unconditionally 0x1: No-idle. An idle request is never acknowledged 0x2: Smart-idle. Acknowledgement to an idle request is given based on the internal activity of the module
Returns
None
