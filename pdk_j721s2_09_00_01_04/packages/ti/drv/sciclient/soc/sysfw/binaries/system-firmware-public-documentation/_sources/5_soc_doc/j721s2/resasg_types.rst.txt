================================================================
J721S2 Board Configuration Resource Assignment Type Descriptions
================================================================

Introduction
============

This chapter provides information of Board Configuration resource assignment
type IDs that are permitted in the J721S2 SoC.  The resource type IDs represent
J721S2 resources ranges assignable to SoC processing entities (or PEs).


**WARNING**: System Firmware RM currently supports a maximum of 442 RM board
configuration resource assignment ranges on the J721S2 SoC.  Sending more
entries than the maximum will result in the RM board configuration being NACK'd
