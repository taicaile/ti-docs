J721S2 Ring Accelerator Device Descriptions¶

Introduction¶

This chapter provides information on the Ring Accelerator devices in the J721S2 SoC. Some System Firmware TISCI messages take device specific inputs. This chapter provides information on the valid values for Ring Accelerator TISCI message parameters.

Ring Accelerator Device IDs¶

Some System Firmware TISCI message APIs require the Ring Accelerator device ID be provided as part of the request. Based on J721S2 Device IDs these are the valid Ring Accelerator device IDs.

Ring Accelerator Device Name Ring Accelerator Device ID
J721S2_DEV_NAVSS0_BCDMA_0 225
J721S2_DEV_NAVSS0_RINGACC_0 259
J721S2_DEV_MCU_NAVSS0_RINGACC0 272

Ring Accelerator Ring Indices¶

This section describes valid Ring Accelerator ring indices for each ring type. The ring index and type ID are used in some Ring Accelerator based TISCI messages.

Warning

Resources marked as reserved for use by TIFS cannot be assigned to a host within the RM Board Configuration resource assignment array. The RM Board Configuration is rejected if an overlap with a reserved resource is detected.
