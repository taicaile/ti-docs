===============================================================
AM64X Board Configuration Resource Assignment Type Descriptions
===============================================================

Introduction
============

This chapter provides information of Board Configuration resource assignment
type IDs that are permitted in the AM64X SoC.  The resource type IDs represent
AM64X resources ranges assignable to SoC processing entities (or PEs).


**WARNING**: System Firmware RM currently supports a maximum of 360 RM board
configuration resource assignment ranges on the AM64X SoC.  Sending more entries
than the maximum will result in the RM board configuration being NACK'd
