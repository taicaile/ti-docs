J721S2 Processor Descriptions¶

Introduction¶

This chapter provides information of Processor and Host IDs that are permitted in the J721S2 SoC. These Processor IDs represent an actual physical processor on the SoC.

NOTE: This should not be confused with HOST_ID.

Enumeration of Processor IDs¶
