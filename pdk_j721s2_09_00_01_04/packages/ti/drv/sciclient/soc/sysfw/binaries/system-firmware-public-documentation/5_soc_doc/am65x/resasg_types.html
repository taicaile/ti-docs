AM65X_SR1 Board Configuration Resource Assignment Type Descriptions¶

Introduction¶

This chapter provides information of Board Configuration resource assignment type IDs that are permitted in the AM6 SoC. The resource type IDs represent AM6 resources ranges assignable to SoC processing entities (or PEs).

WARNING: System Firmware RM currently supports a maximum of 260 RM board configuration resource assignment ranges on the AM6 SoC. Sending more entries than the maximum will result in the RM board configuration being NACK’d
