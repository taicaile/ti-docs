===========================
AM62AX Devices Descriptions
===========================

.. _soc_doc_am62ax_public_devices_desc_intro:


Introduction
============

This chapter provides information on Device IDs that are permitted in the am62ax
SoC.  The device IDs represent SoC subsystems that can be modified via DMSC
TISCI message APIs.  Some Secure, Power, and Resource Management DMSC subsystem
TISCI message APIs define a device ID as a parameter allowing a user to specify
management of a particular SoC subsystem.
