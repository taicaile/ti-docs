================================
J721S2 Secure Proxy Descriptions
================================

.. _soc_doc_j721s2_public_sproxy_desc_intro:

Introduction
============

This chapter provides information of Secure Proxies and communication paths
that are permitted in the J721S2 SoC. These host IDs represent processing entities
(or PEs) which is mandatory identification of a Host in a processor. See
PE/Host documentation for further information

.. _soc_doc_j721s2_public_sproxy_desc_sproxy_list:

=============================

|           1 | MCU_NAVSS0_SEC_PROXY0   |
