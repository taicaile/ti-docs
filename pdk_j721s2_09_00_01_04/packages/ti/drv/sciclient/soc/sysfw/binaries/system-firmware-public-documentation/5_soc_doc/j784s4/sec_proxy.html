J784S4 Secure Proxy Descriptions¶

Introduction¶

This chapter provides information of Secure Proxies and communication paths that are permitted in the J784S4 SoC. These host IDs represent processing entities (or PEs) which is mandatory identification of a Host in a processor. See PE/Host documentation for further information
