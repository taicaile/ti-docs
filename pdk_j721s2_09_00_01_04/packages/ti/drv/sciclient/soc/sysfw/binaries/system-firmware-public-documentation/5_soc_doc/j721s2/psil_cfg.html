J721S2 PSI-L Device Descriptions¶

Introduction¶

This chapter provides information on the PSI-L devices in the J721S2 SoC. Some System Firmware TISCI messages take device specific inputs. This chapter provides information on the valid values for PSI-L TISCI message parameters.

PSI-L Proxy Device IDs¶

Some System Firmware TISCI message APIs require the PSI-L Proxy device ID be provided as part of the request. Based on J721S2 Device IDs these are the valid PSI-L Proxy device IDs.

PSI-L Proxy Device Name PSI-L Proxy Device ID
J721S2_DEV_NAVSS0 224
J721S2_DEV_MCU_NAVSS0 267
