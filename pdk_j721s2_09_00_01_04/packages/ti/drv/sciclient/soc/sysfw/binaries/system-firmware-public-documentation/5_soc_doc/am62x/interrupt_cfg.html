AM62X Interrupt Management Device Descriptions¶

Introduction¶

This chapter provides information on the Interrupt Management devices in the AM62X SoC. Some System Firmware TISCI messages take device specific inputs. This chapter provides information on the valid values for Interrupt Management TISCI message parameters.

Interrupt Router Device IDs¶

Some System Firmware TISCI message APIs require the Interrupt Router device ID be provided as part of the request. Based on AM62X Device IDs these are the valid Interrupt Router device IDs.
