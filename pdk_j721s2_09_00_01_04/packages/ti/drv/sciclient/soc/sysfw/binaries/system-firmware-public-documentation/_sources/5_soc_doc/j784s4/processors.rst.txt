=============================
J784S4 Processor Descriptions
=============================

.. _soc_doc_j784s4_public_proc_desc_intro:


Introduction
============

This chapter provides information of Processor and Host IDs that are permitted
in the J784S4 SoC. These Processor IDs represent an actual physical processor
on the SoC.

NOTE: This should not be confused with HOST_ID.

.. _soc_doc_j784s4_public_proc_desc_processors_list:
