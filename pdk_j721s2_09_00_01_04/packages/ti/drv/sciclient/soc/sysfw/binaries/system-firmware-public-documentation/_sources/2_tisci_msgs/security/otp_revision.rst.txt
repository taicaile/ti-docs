===========================================
OTP Revision Read/Write Message Description
===========================================

.. _sec_otp_rev_rd_wr_message_desc:

This chapter provides information on |sysfw| APIs for reading and writing 
Software Revision and Key Revision to/from eFuses.

The available APIs are listed below.

+------------------+-----------------------------------------------------------------------+-----------------+--------------------+
| TISCI Message ID |                        Message Name                                   | Available in GP | Avialable in HS-FS |
+==================+=======================================================================+=================+====================+
| 0x9032           | :ref:`TISCI_MSG_WRITE_SWREV          <sec-api-wr-swrev-otp>`          | No              | No                 |
+------------------+-----------------------------------------------------------------------+-----------------+--------------------+
| 0x9033           | :ref:`TISCI_MSG_READ_SWREV           <sec-api-rd-swrev-otp>`          | No              | No                 |
+------------------+-----------------------------------------------------------------------+-----------------+--------------------+
| 0x9034           | :ref:`TISCI_MSG_READ_KEYCNT_KEYREV   <sec-api-rd-keycnt-keyrev-otp>`  | No              | No                 |
+------------------+-----------------------------------------------------------------------+-----------------+--------------------+
| 0x9035           | :ref:`TISCI_MSG_WRITE_KEYREV         <sec-api-wr-keyrev-otp>`         | No              | No                 |
+------------------+-----------------------------------------------------------------------+-----------------+--------------------+
