Detailed Description

Place holder for initial data matrices. From this data matrix Pseudo AtA and Pseudo Atb matrices are formed. After forming Pseudo matrices this data is not used.

matrixA : Initial data matrix 'A' formed from track data and projection matrices matrixb : Initial data matrix 'b' formed from track data and projection matrices

Parameters
matrixP_AtA: Pseudo matrix to calculate final AtA
matrixP_Atb: Pseudo matrix to calculate final Atb
