FFTLIB_ifft1d_i32fc_c32fc_o32f
C7x Only Fast Fourier Transform (FFT) kernels

Introduction

Kernel for computing 32-bit floating-point IFFT.

  • The kernel performs IFFT of an N-point complex sequence using N/2-point complex IFFT to output a real sequence, which saves compute time
  • The size of data is required to be a power of 2.
  • The kernel requires application to provide a pre-generated twiddle factors along with the input data.
  • The kernel requires application to provide a pre-generated split factors along with the input data.
  • Supported sizes: 128 to 8192
