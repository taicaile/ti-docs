445 /* \brief Structure used for specifying register configurations of sensor, serializer and deserializer.
446  *
447  * desCfgPreScript : Deserializer config script pre-sensor initialization
448  * serCfgPreScript : Serializer config script pre-sensor initialization
449  * sensorCfgPreScript : Sensor config script without starting streaming
450  * desCfgPostScript : Deserializer script post-sensor initialization
451  * serCfgPostScript : Serializer config script post-sensor initialization
452  * sensorCfgPostScript : Sensor config script after starting streaming
453  *
454  *
455  * Each script is an array of type I2CParams
456  * The scripts maybe of variable lengths. Scripts can also be NULL is the config is not needed.
457  *
