Kernels » HWA OpenVX kernels » TI VPAC VISS (Vision Imaging Sub System) Kernel APIs

Detailed Description

H3A AEW configuration data structure used by the TIVX_KERNEL_VISS kernel.

This is included in the tivx_h3a_data_t H3A output if the AEWB mode of H3A was enabled. This is included in case the 2A algorithm needs to know the configuration that the VISS used for the H3A AEWB engine.
