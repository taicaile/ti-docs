7. Data Sheet¶

Datasheet section contains performance information about various modules within Processor SDK RTOS J721S2.

Important

  • Some datasheet information is not available here and users must request access to the ADDON package for Processor SDK RTOS J721S2 in order to get this information.

  • Users can request access to the ADDON package from here [LINK]

  • Once access is granted, users can download the the ADDON package from here [LINK]

  • Once the ADDON package is installed, the datasheet can be found here ${PSDK_RTOS_ADDON_PATH}/psdk_rtos/docs/datasheet/index.html

Note

All performance measurements provided in this document are PRELIMINARY and have not necessarily been fully optimized yet.
