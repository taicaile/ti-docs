Processor SDK RTOS J721S2¶

Welcome¶

This documentation covers the Texas Instruments Processor SDK RTOS for the J721S2 platform. The following table lists the product devices associated with this platform:

Platform

Devices

J721S2

TDA4VL, TDA4AL, TDA4VE, AM68A

Please refer to the Getting Started page for an overview of dependencies and entry point into the different components and demos available.

Please refer to SDK Components for a complete listing of components included in this platform package.

Important

To run many of the demos in this SDK, the companion Processor SDK Linux for J721S2 also needs to be downloaded separately.
