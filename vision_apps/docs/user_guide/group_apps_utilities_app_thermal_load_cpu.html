Introduction

This application can be used to load C7x, C6x, R5F ranging from 1-99% for a specifed time in seconds. The demo takes three input paramters. core, load in percentage, timer in seconds.

Core Value
C7x 2
MCU2 4
C6x 5
MCU3 6
MCU4 7

Supported plaforms

Platform Linux x86_64 Linux+RTOS mode QNX+RTOS mode SoC
Support NO YES YES J721e / J721S2 / J784S4

Steps to run the application on J7 EVM (Linux + RTOS mode)

  1. Build the application and related libraries as mentioned in Build Instructions for Linux+RTOS mode
  2. Run the app as shown below: The below example loads C7x(Core - 2) at 80% for 5 seconds.
    cd /opt/vision_apps
    ./vx_app_load_test.out 2 80 5
  3. Output will be sent to the terminal standard output.
