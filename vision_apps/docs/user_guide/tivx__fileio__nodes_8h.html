vx_array  in_array,
vx_array  file_path,
vx_array  file_prefix 
)

Creates a vx_array file write Node.

Node accepts a vx_array and optional parameters of path and file name prefix and writes the contents of the vx_array to a file.

Parameters
[in]graphThe reference to the graph.
[in]in_arrayinput array
[in]file_pathpath where file will be written
[in]file_prefixprefix to the file name
Return values
vx_nodeA node reference. Any possible errors preventing a successful creation should be checked using vxGetStatus
